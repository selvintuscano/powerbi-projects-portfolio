# 🏷️ Adidas Sales Analysis

This Power BI dashboard analyzes Adidas' sales performance across products, regions, time periods, and retail channels. It uncovers trends in customer purchasing behavior and highlights top-performing markets, products, and retailers to drive strategic decisions.

## 📊 Key Features
📅 **Monthly Sales Trend**: Visual line chart showing monthly sales fluctuation.  
📦 **Top Products**: Sales breakdown by product category.  
📍 **Sales by State**: Bar chart displaying state-wise sales distribution.  
🌎 **Sales by Region**: Donut chart showing percentage sales contribution by region.  
🛒 **Retailer Performance**: Ranked view of sales through major retail partners.  
📈 **Performance KPIs**: Total sales, operating profit, units sold, price per unit, and operating margin.  
🎛️ **Dynamic Filters**: Region-level dropdown filter to drill into location-specific insights.

---

## 🧠 Skills Demonstrated
🔄 **Data Shaping in Power Query (M)**: Cleaned sales records, handled nulls, grouped monthly sales, and normalized inputs.  
🧮 **DAX Measures**: Operating margin, price per unit, state-wise and product-wise aggregations.  
🧭 **Visual Composition**: Line charts, bar charts, donut charts, KPI cards, slicers.  
🧠 **Business Analysis**: Region-product-retailer interactions and profitability tracking.  
🎨 **Dashboard Design**: Clean layout, color consistency, optimized tooltips and labels.

## 🚀 How to Use
1. Open the file `Adidas-Sales-Analysis.pbix` in **Power BI Desktop**.  
2. Use the **Region filter** in the top-right corner to slice charts by geography.  
3. Hover over visual elements to see detailed tooltips.  
4. Interact with product, retailer, or state bars to drill down into insights.

---

## 🧰 Tech Stack
- Power BI Desktop
